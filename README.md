# A react boilerplate project featuring webpack 2, HMR, code splitting, mobx, SSR

## Installation and Starting

### Development server
```
yarn
yarn start
> visit localhost:3000
```

### Building for production
```
yarn run build
```

### Building for node (server-side rendering)
```
yarn run build-node
```

## Stack

* React
* Mobx
* Webpack 2
* Typescript
* PostCSS + CSS Modules
